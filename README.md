# Noah Site Publisher Assets

Static release assets generated by the Magic Site Publisher workflow.

Live Squarespace pages are not edited from this repository. The one-time Squarespace bootstrap, when installed, reads the promoted `site/latest.json` pointer and externally hosted immutable release assets.

## Website content source

Editable homepage copy lives in [Noah Lucas Website CMS](https://app.notion.com/p/39a602fa891b81f28eddccee12372b0d).

- `Website Settings` contains the hero, Methods, Skills, Expertise, and CTA copy.
- `Employer Profiles` contains one-liner, role, dates, JTBD, workflows, impact, company URL, and display order.
- `site/content/website-content.json` is the published snapshot consumed by staging.
- `scripts/sync-notion-content.mjs` refreshes that snapshot with a Notion integration token.

Run the sync with `NOTION_TOKEN` configured:

```sh
node scripts/sync-notion-content.mjs
```
